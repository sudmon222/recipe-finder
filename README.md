# 🍽️ Recipe Finder

A modern and responsive **Recipe Finder Web App** built using **HTML, CSS, and JavaScript**, allowing users to search for meals, explore random recipes, and view detailed cooking instructions using a public API.

---

## 🚀 Features

* 🔍 Search recipes by name
* 🎲 Get a random recipe instantly
* 📋 View ingredients and step-by-step instructions
* 🎥 Watch recipe videos (YouTube links)
* 📱 Fully responsive design
* ⚡ Fast and clean UI

## 💡 How It Works

* Users enter a recipe name or click the random button
* Data is fetched from TheMealDB API
* Recipes are displayed dynamically
* Clicking a recipe shows full details in a modal
